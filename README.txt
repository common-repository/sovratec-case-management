Manage User requests and use simple workflow to work through each request

== Description ==

This plugin allows wordpress website to track and manage requests  

<h3>Key Features</h3>

The plugin is packaged with the following features:

* Users initiate case menagement by submitting requests.  
* requests go through workflow with each request assigned to right personnel 
* Each step is tracked with date is recorded.  The plugin user perspective and admin perspective.  
* User will receive notification email when request is completed.
* Website admin is able to assign request to right personnel based on case type
